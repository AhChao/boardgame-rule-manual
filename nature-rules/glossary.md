# Nature 遊戲術語對照表 (Glossary)

| 英文術語 (English) | 繁體中文翻譯 (Traditional Chinese) | 說明 (Description) |
| :--- | :--- | :--- |
| Nature | 自然 | 遊戲名稱。 |
| Population | 族群 | 代表族群密度，是主要的計分來源（生存者得 1 分）。 |
| Watering Hole | 水坑 | 遊戲中央區域，放置植物性食物。 |
| Plant Food | 植物性食物 | 由水坑獲取的綠色食物。 |
| Meat Food | 肉類性食物 | 透過捕食獲得的紅色食物。 |
| Size Dial | 體型轉盤 | 追蹤物種體型的工具。體型越大，覓食量越多，防護力也越強。 |
| Trait | 特性 | 賦予物種特殊能力的卡牌（如：Fast, Scavenging）。 |
| Hunter (Carnivore) | 獵食者 (肉食性) | 裝備獵食者特性的物種，不再從水坑進食植物，而是捕獵其他動物。 |
| Herbivore | 草食性 | 基本狀態，從水坑進食植物。 |
| Omnivore | 雜食性 | 擁有觸發性肉食能力的物種，可同時進食植物與肉類。 |
| Adapting Phase | 適應階段 | 玩家出牌增加族群、體型或賦予特性的回合。 |
| Feeding Phase | 覓食階段 | 物種輪流進食植物或進行捕獵。 |
| Preparation | 準備階段 | 增加水坑食物、獲得新物種、抽取卡牌。 |
| Scoring | 計分階段 | 將進食的食物存入屏風後，計算分數。 |
| Starvation | 飢餓 | 當食物不足以支撐族群時，導致族群減少的情况。 |
| Predation / Hunt | 捕食 / 狩獵 | 獵食者攻擊其他物種以獲取肉類食物。 |
| Extinction | 絕種 | 當一個物種的所有族群都被移除時。 |
| Defense Value | 防護值 | 物種抵禦捕食的能力，等於體型加上特性修正。 |
| Hunt Value | 攻擊值 | 獵食者發起捕食的能力，等於體型加上特性修正。 |
| Trigger Trait | 觸發性特性 | 帶有驚嘆號（!）的特性，在條件滿足時自動執行。 |
