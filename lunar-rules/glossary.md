# Lunar (月色之境) Glossary

| English | Traditional Chinese | Description |
| :--- | :--- | :--- |
| Partnership Tiles | 隊伍板塊 | 每隊持有的板塊，標示誰具有「帶牌 (Lead)」權。 |
| Trick Tracking Board | 吃墩計分板 | 用於追蹤每墩勝負的板塊。 |
| Trick Tracking Markers | 吃墩標記 | 在計分板上移動的標記。 |
| Scoreboard | 總計分板 | 記錄每輪結束後的總分。 |
| Trump Tiles | 王牌板塊 | 用於決定當前輪次王牌花色的板塊。 |
| Rank Cards | 數字牌 | 點數 1-8 的牌。 |
| Suit Cards | 花色牌 | 狼、貓頭鷹、山貓、蝙蝠四種花色。 |
| Wolf | 狼 | 花色之一。 |
| Owl | 貓頭鷹 | 花色之一。 |
| Lynx | 山貓 | 花色之一。 |
| Bat | 蝙蝠 | 花色之一。 |
| Lead | 帶牌 / 出牌 | 一墩中第一位出牌的行為或權利。 |
| Lead Suit | 帶牌花色 | 該墩第一張被打出的花色，後續玩家需跟隨。 |
| Trump Suit | 王牌花色 | 該輪中最強的花色。 |
| Trick | 墩 | 玩家各出一張牌後組成的回合。 |
| Sky | 天空區 | 二人遊戲中，翻開在桌面供選擇的公用牌區。 |
| Table Talk | 桌面交談 | 遊戲進行中關於手牌或策略的討論（通常禁止）。 |
| Round | 輪 | 手牌用完為止的一個大回合。 |
| Match | 局 | 達到目標分數（30分）結束整場遊戲。 |
