# Molly House 術語對照表 (Glossary)

## 核心概念 (Core Concepts)

| 英文術語 (English) | 繁體中文翻譯 (Traditional Chinese) | 說明 (Description) |
| :--- | :--- | :--- |
| **Molly House** | **摩立屋** | 18 世紀倫敦的地下社群集會所。 |
| **Molly / Mollies** | **摩立** | 社群成員，性別與性取向的開拓者。 |
| **Society for the Reformation of Manners** | **風紀改良協會** | 試圖取締「偏差行為」的市民團體，本遊戲的對抗力量。 |
| **Joy** | **喜悅** | 遊戲中的主要資源與勝利分數。 |
| **Community Joy** | **社群喜悅** | 衡量社群整體存續的指標。 |
| **Gossip Pile** | **流言堆** | 蓋放的棄牌堆，會積累證據並引發搜查。 |
| **Safe Pile** | **安全堆** | 開放的棄牌堆，不會對玩家造成傷害。 |
| **Evidence** | **證據** | 針對特定房屋積累的負面標記。 |
| **Reputation** | **聲望** | 玩家在各花色中累積的卡片與方塊。 |
| **Informer** | **線人** | 背叛社群、向協會通風報信的玩家。 |
| **Exposure / Exposed** | **暴露** | 當流言引發威脅時，玩家必須失去相應聲望。 |
| **Most Infamous** | **最惡名昭彰** | 在特定花色中聲望最高的玩家。 |

## 卡片類型 (Card Types)

| 英文術語 (English) | 繁體中文翻譯 (Traditional Chinese) | 說明 (Description) |
| :--- | :--- | :--- |
| **Vice Cards** | **墮落卡** | 遊戲中的主要卡牌。 |
| **Desires** | **慾望** | 數字卡，主要用於增加聲望與慶典得分。 |
| **Threats** | **威脅** | 包含治安官與無賴，會危害社群。 |
| **Constables (C)** | **治安官** | 一種威脅卡，會干擾慶典。 |
| **Rogues (R)** | **無賴** | 一種威脅卡，數值多變。 |
| **Mollies (J/Q)** | **摩立卡** | 傑克 (Jack) 與皇后 (Queen)，提供加分。 |
| **Item Cards** | **物品卡** | 在商店購買的輔助道具。 |
| **Indictment** | **起訴** | 房屋遭搜查後，對聲望最高者的懲罰。 |

## 行動 (Actions)

| 英文術語 (English) | 繁體中文翻譯 (Traditional Chinese) | 說明 (Description) |
| :--- | :--- | :--- |
| **Lie Low** | **隱忍** | 從牌堆抽牌。 |
| **Indulge** | **沉溺** | 從市場取得符合所在地花色的卡。 |
| **Accuse** | **指控** | 揭露另一名玩家的身份標記。 |
| **Cruise** | **遊蕩** | 增加聲望並從牌堆抽牌。 |
| **Shop** | **購物** | 取得物品卡。 |
| **Throw a Festivity** | **舉辦慶典** | 主要得分行動。 |

## 慶典等級 (Festivity Ranks)

| 英文術語 (English) | 繁體中文翻譯 (Traditional Chinese) | 說明 (Description) |
